## 项目定位

Go-Ai-Studio 面向短视频内容生产者、AI 视频工作流玩家、剧情号运营者、探店博主与本地化内容团队，目标是把「写脚本、拆分镜、生成角色、生成场景、生成视频、生成配音、整理导出」这些分散步骤，收束到一个可视化工作台里。

它不是单纯的提示词工具，而是一个围绕内容成片流程设计的生产系统：从项目规划到素材生成，从多集连续剧情到口播讲解，从博主探店到音频生产，都尽量用一键化、批量化和可追踪的方式完成。

## 适合做什么

- **AI 短剧连续剧**：角色设定、集数延续、分镜规划、图片与视频批量生成。
- **小说/故事视频化**：把长文本拆成更适合出图、视频和配音的镜头结构。
- **本地生活探店**：围绕门店、菜品、空间、博主视角生成成套内容。
- **口播讲解视频**：从一段文案规划成带讲解人、素材镜头和转场的视频方案。
- **批量配音生产**：按角色、按人设、按提示批量生成音频资产。
- **视觉资产扩展**：基于参考图生成角色、场景、局部特写和多角度素材。

## LLM 引擎建议

LLM 引擎的质量会明显影响最终效果，尤其是：

- 剧情是否连贯
- 分镜是否合理
- 角色设定是否稳定
- 图片提示词是否准确
- 视频动作描述是否可执行
- 配音台词是否自然
- 探店与讲解内容是否有真实商业表达

作者个人建议优先使用 **claude-sonnet-4-6**。如果你暂时找不到国内可用方式，可以尝试这个 API 中转：

**注册链接**：https://yinli.one/register?aff=3J2w

在 Go-Ai-Studio 的 **LLM 引擎** 中可以这样配置：

- 接口地址：`https://yinli.one/v1`
- 模型名称：`claude-sonnet-4-6`

当然，你也可以根据自己的预算、速度和稳定性，配置其他兼容接口的大模型服务。建议优先选择长文本理解能力强、中文表达稳定、结构化输出可靠的模型。

## 产品亮点

### 🎞️ 从脚本到镜头

不是只帮你写一段文案，而是进一步拆成角色、场景、镜头、画面提示词、视频动作提示词和可追踪的任务状态，让内容真正进入生产流程。

### 🧩 多模式内容生产

同一个工作台里覆盖短剧、探店、综合讲解、多视觉图、配音与语音克隆，适合把不同类型的 AI 内容工作流沉淀成统一流程。

### 🧭 可视化项目管理

每个项目都有独立的素材、状态和生成进度，方便查看哪些内容已完成、哪些需要重试、哪些可以导出。

### 🎨 画风与模型选择

支持管理画风、选择默认图片模型和视频模型，让不同项目可以保持更稳定的视觉方向。

### 📦 双击即用的发行包

Release 版本会提供 Windows、macOS、Linux 以及 amd64/arm64 架构包。下载解压后运行程序，即可打开本地工作台。

## 推荐工作流

### AI 短剧

1. 创建剧情项目
2. 选择画风与生成标签
3. 输入故事方向或分集要求
4. 一键生成角色、分镜、场景与视频素材
5. 检查结果并批量重试或导出

### 配音生产

1. 创建配音项目
2. 选择按人设生成或按提示生成
3. 输入脚本或导入拆分结果
4. 批量生成音频
5. 导出整套配音资产

### 博主探店

1. 创建探店项目
2. 录入门店、场景、菜品或素材参考
3. 规划探店镜头与展示重点
4. 生成图片、视频与导出结果
5. 用于本地生活、餐饮推广或门店展示
